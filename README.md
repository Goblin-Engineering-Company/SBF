# Single-Button Fishing (SBF)

**The entire fishing loop on one key.** Cast, interact, loot, re-buff, swap
gear, and keep a full journal of every catch — without touching your bags.

Built by the [Goblin Engineering Company](https://goblineng.co).

## Features

- One-key loop: cast, interact, loot — with combat and self-heal fallbacks
  built in
- Slot system for food, drink, bobbers, lures, pole enchants, chum, wards, and
  boats
- Automatic buff upkeep: knows durations and never wastes a reapplication
- Per-character and account-wide (Warband) profile scopes
- Fish log: every catch recorded with zone and map position
- Stats: all-time and per-session rollups, fish × zone breakdowns
- Welcome screen sets up a new character in under a minute

## Install

1. Download the latest release and copy the `SBF/` folder into
   `World of Warcraft/_retail_/Interface/AddOns/`.
2. Enable **SBF** in the AddOns list and log in (or `/reload`).

## Quick start

- Open the welcome screen (`/sbf welcome`), bind your cast key, and go fish.
- The options window (`/sbf`) holds the slot setup, buff settings, and the
  fish log.
